## Tasks

Below are the tasks requirements and examples of input data to be provided for the different programs and expected outputs.

#### 1. CountRealNumbers

Test inputs and outputs:

```
Input:
3 2 5 4.5 8 2 3 5 9

Output:
2.0 -> 2
3.0 -> 2
4.5 -> 1
5.0 -> 2
8.0 -> 1
9.0 -> 1
```
