# Java Fundamentals

## Primitive types

- byte
- short
- int
- long
- float
- double
- char
- boolean

```java
byte b = 23;
short s = 21000;
int i = 45000;
long l = 12345678912L;

float f = 4.5f;
double d = 3.42345234234423;

char c = 'b';

boolean bool = true;
```

```java
String s = "hello";
```
