## Tasks

Below are the tasks requirements and examples of input data to be provided for the different programs and expected outputs.

#### 1. DecimalComparator

Write a Java method that determines whether two decimal numbers are equal when compared up to three decimal places. The method should accept two `double` values as parameters.

Test inputs and outputs:

```
Input:
num1 = 3.1756
num2 = 3.175

Output:
true
```

```
Input:
num1 = 3.176
num2 = 3.175

Output:
false
```
