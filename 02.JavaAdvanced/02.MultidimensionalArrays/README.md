1. CompareMatrices
2. PositionsOf
3. IntersectionOfTwoMatrices
4. SumMatrixElements
5. MaximumSumSubmatrix
6. PrintDiagonals
7. FindRealQueen
8. WrongMeasurements

12. MaximalSum
13. MatrixShuffling
14. StringMatrixRotation
15. Crossfire
16. HeiganDance
17. ReverseMatrixDiagonals
