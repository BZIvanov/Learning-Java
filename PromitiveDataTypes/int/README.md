# int data type

## Types

### Binary

Binary numbers use the prefix `0b`

```java
int a = 0b1010; // Value will be 10
```

### Octal

Octal numbers use the prefix `0`

```java
int a = 010; // value will be 8
```

### Hexadecimal

Hexadecimal numbers use the prefix `0x`

```java
int a = 0x10; // value will be 16
```
